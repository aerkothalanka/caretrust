# CareTrust Architecture Diagram

Demo-ready architecture diagram for the CareTrust app.

## Included files

- `caretrust-architecture-diagram.html` — editable, self-contained HTML source.
- `caretrust-architecture-diagram.png` — 1920×1080 exported PNG for slides/demo use.

## Contents

The diagram presents three high-level sections:

1. **Data Layer** — Bronze, Silver, Gold.
2. **Agent Layer** — Trust Scoring, Planner Assistant, Voice + Phone, Human-in-the-Loop.
3. **App Layer** — User Plan Experience, Service Layer, TrustCard Actions.
